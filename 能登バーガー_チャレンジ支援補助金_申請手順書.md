## 申請手順 全ステップ

### STEP 1：申請事業者の確定と要件確認（4月中）

**やること：**
- [ ] 金丸商店株式会社の代表者・決算書の有無を小川氏に確認
- [ ] 金丸商店が「小規模事業者」か「中小企業」かを確認（従業員数で判定）
  - 小規模事業者（商業・サービス業：常時従業員5人以下）→ 補助率2/3（有利）
  - 中小企業 → 補助率1/2
- [ ] 一般社団法人（ヒーローズラボ）が補助対象かを能登事業者支援センターに電話で確認
- [ ] 申請主体を金丸商店に確定した場合、代表者の実印・印鑑証明を取得手続き開始

---

### STEP 2：能登事業者支援センターへの相談（4月中〜5月上旬）

**連絡先：**
- フリーダイヤル：**0120-262-380**（平日10:00〜17:00）
- または補助金事務局：0120-036-682

**電話で伝える内容（トークスクリプト）：**
```
「令和6年能登半島地震後、能登町宇出津で飲食店を経営している
 金丸商店と申します。能登産のタコ・魚介類を使ったハンバーガー
 （能登バーガー）の新商品開発と、キッチンカーを使った石川県内
 外のイベントでの移動販売という新たな事業に取り組みたいと考え
 ております。チャレンジ支援補助金の対象になるか確認したい。
 また事業内容事前確認書の発行をお願いしたい。」
```

**電話で確認するポイント：**
- [ ] 金丸商店（飲食業）→ 能登バーガー製造・移動販売が「新たなチャレンジ」に該当するか
- [ ] 新たな業種・事業・市場のどの類型で申請するか
- [ ] キッチンカーでの石川県外イベント出店が「能登6市町以外への移転」に当たらないか
- [ ] 設立2期未満の場合の決算書代替書類の取り扱い
- [ ] 一般社団法人（ヒーローズラボ）を共同申請者として含められるか

**この段階で発行してもらうもの：**
- 事業内容事前確認書（第2号様式）← これがないと申請できない

### STEP 3：事業計画書の骨子策定（5月中）

米永（CBC）のサポートで以下を作成する。

#### 事業計画書の構成（記載必須項目）

**① 企業概要**
- 金丸商店の事業概要・現在の主力商品・強み
- 震災による経営環境変化（売上減・客数減の具体的数値）

**② 新たなチャレンジの内容**
- 何が「新たな業種・事業・市場」か明確に定義する

```
例）「新たな事業への挑戦」
現業：水産物の仕入・販売（地元客向け）
新事業：能登産タコ・魚介類を使ったハンバーガーの商品開発・製造
        およびキッチンカーによる石川県内外のイベント出店販売
→ 既存の水産仕入ルート（下平鮮魚等）を活かしつつ、
  飲食加工製造・移動販売という新業態に挑戦
```

**③ 市場分析と収支計画**

| 項目 | 数値 |
|------|------|
| 目標販売単価 | 1個 800〜1,200円 |
| 1イベント販売目標 | 50〜100個 |
| 年間イベント出店目標 | 12〜20回 |
| 補助期間終了後の年商目標 | ○○万円 |

※数値はアンケート・他イベント実績等のエビデンスを添付すること

**④ 実施スケジュール**

| 時期 | 内容 |
|------|------|
| 2026年4〜6月 | レシピ開発・試作テスト（能登高校生・カタリバ協力） |
| 2026年7月 | 試食評価・改善 |
| 2026年8〜9月 | キッチンカー仕様確定・発注 |
| 2026年10月〜 | 石川県内イベント初出店 |
| 2027年〜 | 首都圏・関西のイベントへの展開 |

**⑤ 補助対象経費の明細（概算）**

| 経費項目 | 補助金分類 | 概算額 |
|---------|-----------|--------|
| 試作食材費・開発費 | 新商品開発費 | 30万円 |
| パッケージデザイン・ロゴ制作 | 新商品開発費 | 20万円 |
| チラシ・SNS広告・動画制作 | 広告宣伝費 | 30万円 |
| 展示会・イベント出展料・交通費 | 広告宣伝費 | 20万円 |
| キッチンカー内装・改造工事 | 委託・外注費 | 50万円 |
| キッチンカー本体購入 | 車両購入費（上限50万円） | 50万円 |
| 米永・小川のコンサル料 | 専門家経費 | 30万円 |
| 備品購入（1点30万円未満のもの） | 備品購入費 | 20万円 |
| **合計（概算）** | | **250万円** |
| **補助申請額（×2/3）** | | **約167万円** |

**⑥ 連携機関**
- 能登高校（試作・レシピ開発への高校生参加）
- カタリバ（能登高校魅力化プロジェクト）

### STEP 4：必要書類の収集・作成（5月〜7月）

#### 申請者が用意する書類

- [ ] **宣誓・同意書（第1号様式 別紙1）** ← 代表者の自筆署名が必須
- [ ] **役員等名簿（第1号様式 別紙2）**

### STEP 8：採択後の対応

採択通知が届いたら：
1. **交付決定通知書**を受領
2. 事業実施（経費発生）を開始
3. 実績報告書と領収書等を提出
4. 補助金の請求・受領

**注意点：**
- 交付決定前に発生した経費は原則対象外（ただし災害発生日R6.1.1まで遡及適用可能な場合あり）
- 領収書・請求書は全て保存
- 変更が生じた場合は事前に事務局へ連絡

## リスクと対策

| リスク | 対策 |
|--------|------|
| 金丸商店が決算書を用意できない（新設法人） | 支援センターに代替書類の可否を確認。設立時の事業計画書等で代替できる場合あり |
| 「新たなチャレンジ」に該当しないと判断される | 事前確認書発行前に口頭で十分説明し、担当者と方向性を合わせる |
| キッチンカー費用が単体で予算超過 | 内装改造工事（委託外注費）と車両本体（上限50万円）を分けて計上 |
| 7月31日に試作が間に合わない | 9次（9月30日）に後ろ倒しも選択肢。販売実績を添付できる分、採択率が高まる可能性あり |
| CBC（宝塚）が関与しすぎると「外部企業の下請け」と見られる | 米永を「専門家・コンサルタント（謝金）」として位置づけ、主体は金丸商店であることを明確にする |
