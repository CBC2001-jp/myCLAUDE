📍 福井県高浜町 令和7年度 地域再生マネージャー

高浜の水産品を、米国市場へ——対米HACCP認証への挑戦

高浜町には優れた水産加工品がある。しかしその価値は国内市場にとどまっている。この食を世界基準に引き上げれば、町の産業の未来が変わる。

地域再生マネージャーとして、大日本水産会と連携し対米HACCP認証取得プロジェクトを推進中。認証ロードマップの策定から、投資対効果を含む事業計画の立案、加工場改修計画、現場教育プログラムの設計と実行まで一気通貫で伴走します。

目標：2027年 認証取得

現在の進捗

米国の厳格な基準をどうクリアし、どうやって海外マーケットに届けるか。大日本水産会の認証アドバイザーによる現地指導がスタート。対米HACCP認証の取得に向け、現場の土台づくりを一つひとつ着実に進めています。

ゴールまでの設計

事業計画、加工場改修、製造マニュアル、現場教育プログラムを含むロードマップは策定済み。同アドバイザーの指導実績では、計画どおり進めたケースでほぼすべてが認証を取得しています。

✓ 米国基準HACCP認証ロードマップ策定 ✓ 現場教育プログラム設計 ✓ 加工場改修計画立案 ✓ 投資対効果を含む事業計画策定

📍 石川県能登町 令和8年度 観光庁採択

地域観光資源の多言語解説整備促進事業（観光庁 令和8年度 採択）

石川県能登町にて、観光庁「令和8年度 地域観光資源の多言語解説整備促進事業」に採択されました。地域の文化財の真脇遺跡・あばれ祭・食の復興体験などに関する町との調整、および文化コンテンツの現地監修を担当。多言語解説文の制作・実装を、地域・観光・専門家との連携体制で推進します。

✓ 真脇遺跡・あばれ祭・食の復興体験 ✓ 文化コンテンツ現地監修 ✓ 多言語解説文 制作・実装

03
📍 和歌山県みなべ町

デジタル × 一次産業で、関係人口をつくる

世界農業遺産「みなべ・田辺の梅システム」の地で、地元農家と協働し南高梅の木の年間オーナー制度を設計・実施。KDDIのNFTプラットフォーム（αU Market）を活用し、オーナーが梅の木の成長を見守りながら、季節ごとの返礼品や収穫・農泊体験を楽しめる仕組みを構築しました。

デジタル技術とリアルな農業体験を掛け合わせ、県外からの継続的な関係人口を創出。一次産業のサステナブルな経営を支えるモデルとして取り組んでいます。

✓ KDDIのNFTプラットフォームを活用したオーナー制度設計 ✓ 年間を通じた返礼品・体験プログラムの構築 ✓ 県外からの継続的な関係人口の創出
Our Strengths

CBCの強み

町を元気にしたいという想いだけでは、地域は変わらない。想いを形にするには、実務の力が要る。CBCが地域に提供するのは、その実務力です。

食品の現場を知り尽くした30年の実務力

加工場の床の傾斜をどうするかという現場の細部から、海外マーケットへの商談まで。最初から最後まで現場に入って一緒に動きます。

30年間、食品ビジネスの世界で「価値ある商品づくり」と「どうすれば売れるか」を自ら実践してきました。研究開発、製造・品質管理、海外事業運営——そのノウハウを、地域の事業に活かします。

HACCP認証 事業計画立案 対米輸出支援

海外ネットワークで、地域と世界をつなぐ

海外IT企業の意思決定者と直接つながるネットワークを活かし、地域向けインバウンド・観光アプリを企画・提案。現在、具体的なプロジェクトが進行中です。

・海外企業トップとの直接連携 ― 代理店や商社を介さず、意思決定者とダイレクトにつながる
・地域の観光資源を再発掘 ― 海外目線で地域の魅力を見直し、デジタルで届ける
・インバウンド誘客の入口づくり ― 「知る・訪れる・関わる」の流れを設計する

海外企業連携 インバウンド・観光アプリ 関係人口創出

「強み」を活かして、地域のチームを動かす

地域創生の現場で最も難しいのは、事業計画ではなく人間関係とチームづくり。限られた人数で成果を出すには、一人ひとりの「強み」を見つけ、活かし合える関係をつくることが鍵になります。

Gallup認定ストレングスコーチとして、個人の資質を言語化し、チームの力に変えるプログラムを提供。国内大手メーカーのマネジメント層へのワークショップ実績もあります。食品ビジネス30年の現場感覚を持つコーチだからこそ、「分析して終わり」ではなく、事業の推進力に直結する支援ができます。

・個人の資質を見える化し、互いの強みを掛け合わせるチームへ
・現場を知るコーチだから、机上論ではないアドバイスができる
・強みへのフォーカスが、自発性と突破力を生む
